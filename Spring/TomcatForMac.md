# Mac Tomcat 설치

```
//톰켓 버전 확인
brew search tomcat
//원하는 버전으로 설치
brew install tomcat@8
//설치확인
brew list

//Finder
cmd+shift+g => 폴더이동
/usr/local/Cellar/ =>brew에 설치된 파일들을 볼 수 있다.
```

## Tomcat 실행 및 종료

```
cd /usr/local/Cellar/tomcat@8/8.5.43/bin
./catalina start =>실행명령어
./catalina stop =>종료명령어
```
